# Installation

## With Aftman (preferred)
[Aftman][aftman] is the toolchain manager we recommend. You can use it to install gooey:

In your project (preferred)
```bash
aftman init
aftman add UpliftGames/wally
aftman install
```

Or install gooey globally
```bash
aftman add --global UpliftGames/wally
aftman install
```

[aftman]: https://github.com/LPGhatguy/aftman

## Homebrew
[Homebrew][homebrew] is also a convenient way to install gooey! (only for MacOS/Linux)
```bash
brew install gooey
```

[homebrew]: https://brew.sh/

## From GitHub
Pre-built binaries are available for Windows, macOS, and Linux from the [GitHub Releases Page for gooey][releases].

[releases]: https://github.com/UpliftGames/wally/releases

## From Source
It's straightforward to compile gooey from source. gooey requires Rust 1.51.0 or newer.

Clone the repository and use:

```bash
cargo install --locked --path .
```
